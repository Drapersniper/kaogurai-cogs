# kaogurai's cogs

## Installation

```shell
[p]load downloader

[p]repo add kao https://github.com/ryan5453/kaogurai-cogs

[p]cog install kao <cog>

[p]load <cog>
```

## Support

Please make an issue on this GitHub repository if you have any questions or feedback.

| ------------ | ------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| AIArt        | v2.3.2  | Generate art using AI.                                                                                                              |
